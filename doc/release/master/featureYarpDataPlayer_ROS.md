### Tools

#### yarpdataplayer

* `yarpdataplayer` is now able to reproduce several ROS types, i.e.
 ```
sensor_msgs/LaserScan.
nav_msgs/Odometry
tf/tfMessage
tf2_msgs/tfMessage
geometry_msgs/Pose
geometry_msgs/Pose2D
```
Other types can be easily added just editing `utils.cpp` and `worker.cpp`
